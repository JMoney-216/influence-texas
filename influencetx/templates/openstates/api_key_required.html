Open:States API key required

  1. Register for an Open States API key
    • Use your own name and email
    • Website: https://www.open-austin.org/
    • Organization: Open Austin
    • Intended Usage: Local development of influencetx app
  2. You should receive an email with your new API key. Follow the activation link.
  3. Copy your api key into the following command and run it in the root directory of the repo (i.e. where README.md is located):

    echo "OPENSTATES_API_KEY=YOUR-API-KEY-HERE" >> .influencetx-secrets.env
